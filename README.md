# 🚀 Mi Portfolio Profesional - Sergio Dev

Bienvenido a mi portfolio personal. Aquí comparto mi trayectoria, las tecnologías que domino y los proyectos que he desarrollado, pasando desde interfaces dinámicas en el frontend hasta arquitecturas robustas en el backend.

## 👨‍💻 Sobre Mí
Soy un **Desarrollador Fullstack** apasionado por la creación de soluciones digitales completas. Mi enfoque principal es construir aplicaciones web escalables y eficientes, cuidando tanto la experiencia de usuario (UX/UI) como la integridad y seguridad de los datos.

## 🛠️ Stack Tecnológico

### Backend
* **Lenguaje:** Java 17+
* **Framework:** Spring Boot 4.0.3
* **Persistencia:** Spring Data JPA / Hibernate
* **Base de Datos:** MySQL, SQLite

### Frontend
* **Core:** JavaScript (ES6+), HTML5, CSS3
* **Estilo:** Diseño Responsivo, Animaciones con Intersection Observer
* **Herramientas:** Boxicons, Google Fonts

## 📂 Proyectos Destacados

* **Descripción:** Mi proyecto más ambicioso. Un e-commerce completo con gestión de inventario, carrito de compras y API REST funcional.
* **Tecnologías:** Spring Boot, MySQL, JavaScript.

### 🌮 Antojitos Mexican Curious (TPV)
